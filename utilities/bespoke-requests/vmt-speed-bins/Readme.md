These are modified versions of the scripts of the same name found in 
[../../../model-files/scripts/emfac](../../../model-files/scripts/emfac).

These variants were created for CARB staff's request for PBA50+ modeled VMT by speed bins for the [Bay Area Air District's Climate Action Plan](https://www.baaqmd.gov/en/Plans-and-Climate/Climate-Planning/Bay-Area-Regional-Climate-Action-Planning-Initiative); the only difference between these and the source scripts are the modified speed bins.

See [Asana task (internal only): PBA50+ VMT by speed bin for Clean Air Plan (CAP)](https://app.asana.com/1/11860278793487/project/310827677834656/task/1214314337221009?focus=true)
